## Selected Work

### AI Infrastructure

**[recall](https://github.com/AidenGeunGeun/recall)** · [`@skybluejacket/recall`](https://www.npmjs.com/package/@skybluejacket/recall)  
Semantic memory search for AI agents. CWD-scoped — cd into any repo and get coherent cross-session memory. Agents hallucinate when they reconstruct state from compressed context; `recall` grounds them.

**[transcribe-cli](https://github.com/AidenGeunGeun/transcribe-cli)** · [`@opencode/transcribe-cli`](https://www.npmjs.com/package/@opencode/transcribe-cli)  
Local document-to-markdown OCR for researchers. PDFs, DOCX, XLSX, HWP, images — all offline, no cloud. Routes clean text through fast paths and math-dense pages through GLM-OCR. Suspicious output is flagged, not silently trusted.

**[code-intel](https://github.com/AidenGeunGeun/code-intel)**  
Structural codebase index engine with three retrieval paths: `search` for compact entity lookup, `trace` for bounded relationship paths, and `gather` for evidence retrieval before reading files.

**[exa-cli](https://github.com/AidenGeunGeun/exa-cli)**  
Three-tier web research via the Exa Search API. `auto`, `synthesis`, and `deep` tiers for everything from quick fact checks to structured multi-source extraction.

**[todoist-cli](https://github.com/AidenGeunGeun/todoist-cli)**  
JSON-first Todoist CLI for agent task, project, section, and user workflows. Headless automation without the web UI.

**[OpenCodeOrchestra](https://github.com/AidenGeunGeun/OpencodeOrchestra)**  
Multi-layer agent orchestration for OpenCode. PM plans, orchestrator executes, specialists investigate/review/document. Each role has its own depth, permissions, and model. Fork of OpenCode v1.2.5.

**[opencode-context-compress](https://github.com/AidenGeunGeun/opencode-context-compress)**  
Manual-first context compression plugin. No autonomous loops, no per-turn nudges. Compression runs only when you trigger `/compress manage`. Agent decides what to fold and how tersely.

### Aerospace

**[PINN_Guidance](https://github.com/AidenGeunGeun/PINN_Guidance)**  
Physics-informed neural networks for air-to-air missile guidance. Classical 3-DOF simulator + differentiable physics / PMP stack + PINN rebuild. Learning costate and final time, computing control from PMP structure, rolling out state through differentiable physics. Python + MATLAB.

**[hvt-missile-sim](https://github.com/AidenGeunGeun/hvt-missile-sim)**  
High-Value Target missile defense simulation in MATLAB. Intercepts ballistic targets using multiple guidance algorithms — PNG, APN, and variants. Parameterizable threat profiles and interceptor configs.

**[6dofsim](https://github.com/AidenGeunGeun/6dofsim)**  
Six-degree-of-freedom missile simulator in MATLAB. Full rigid-body dynamics with aerodynamic lookup tables.

**[Coop_guidance](https://github.com/AidenGeunGeun/Coop_guidance)**  
Cooperative guidance law implementation in Python. Multi-agent interception with shared information.

**[9M723-Iskander-missile-trajectory](https://github.com/AidenGeunGeun/9M723-Iskander-missile-trajectory)**  
Aeroballistic rocket trajectory modeling and analysis. Jupyter Notebook.
